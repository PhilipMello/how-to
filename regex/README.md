## REGEX | EXPRESSIONS

Date EN:

```
[0-9]{4}[-][0-9]{2}[-][0-9]{2}
```

Email:

```
([a-zA-Z0-9]+)@([a-zA-Z0-9]+)\.([a-zA-Z0-9]+)([.][a-zA-Z0-9]+)?
```

Email with filter by providers:

```
[a-zA-Z0-9_]+[@](hotmail|live|gmail|msn|outlook)[.][a-zA-Z0-9_]+([.][a-zA-Z]+)?
```

Portugal NIF:

```
[0-9]{9}
```

Brazilian CPF:

```
[0-9]{3}[.][0-9]{3}[.][0-9]{3}[-][0-9]{2}
```
